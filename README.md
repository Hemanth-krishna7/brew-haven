# Brew Haven

Brew Haven is a modern cafe website built to showcase a premium coffee shop with a clean and welcoming design. The website focuses on creating a smooth browsing experience while presenting the cafe's menu, story, gallery, and reservation details.

## Live Demo

https://brew-haven-gilt.vercel.app/

## What You'll Find

- Responsive design for mobile, tablet, and desktop
- Elegant landing page
- Menu with food and beverage items
- About section introducing the cafe
- Image gallery
- Reservation form with client-side validation
- Smooth scrolling navigation

## Built With

- React
- Vite
- Tailwind CSS
- JavaScript

## Run Locally

```bash
git clone https://github.com/Hemanth-krishna7/brew-haven.git
cd brew-haven
npm install
npm run dev
```

## About This Project

Brew Haven is a modern cafe website that gives visitors a simple way to explore the cafe, browse its menu, view the ambience through a gallery, and reserve a table. The website is designed with a clean layout and adapts smoothly across mobile, tablet, and desktop devices.
